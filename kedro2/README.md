# Setup

```
py -3.11 -m venv .venv
./.venv/Scripts/activate
python -m pip install --upgrade pip
pip install -r ./requirements.txt
```

# Run

```
kedro run
```

# Visualization

```
kedro viz run
```

Open [http://localhost:4141](http://localhost:4141).
